# Expense Tracker (Flutter)

A simple personal expense tracker built with Flutter.  
It uses MVVM and Provider, stores data locally with Drift (SQLite).


## Features

- Add/delete expenses.
- Insights and charts
- Dark/Light theme.
- Currency selection (saved locally).
- Local storage with Drift.
